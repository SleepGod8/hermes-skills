# 世界征服者4 自动扩张 - 游戏逻辑参考

## 征服模式流程

1. 进入征服模式 → 选择年代（1939/1943/1950/1980）
2. 选择国家 → 进入大地图
3. 每回合操作：
   - 点城市 → 生产部队（坦克优先，步兵其次）
   - 移动部队向前线
   - 攻击范围内敌人
   - 结束回合
4. 重复直到统一或失败

## 常见问题

### ADB 连不上
- MuMu 12 默认端口可能是 `16384` 或 `21503`
- 检查：MuMu → 设置 → 其他 → 开启 ADB 调试
- 终极方法：`adb kill-server && adb start-server && adb connect 127.0.0.1:16384`

### 模板匹配不到
- 模板截图必须和模拟器分辨率一致
- 尝试降低 threshold（0.7 → 0.6）
- 按钮可能被 UI 动画遮挡，增加等待时间

### 游戏卡住
- 可能是弹窗挡住了，脚本会自动处理常见弹窗
- 5 回合无行动会自动停止
- 手动检查游戏状态
